1. Trochę słów o JavaScript

JavaScript jest dynamicznym, interpretowanym językiem programowania, który jest powszechnie stosowany do tworzenia interaktywnych stron internetowych i aplikacji internetowych. Jest to jedna z trzech głównych technologii używanych w tworzeniu stron internetowych, obok HTML (Hypertext Markup Language) do tworzenia struktury i treści strony oraz CSS (Cascading Style Sheets) do określania wyglądu i stylów stron. JavaScript jest językiem skryptowym, co oznacza, że jest wykonywany w przeglądarkach internetowych klienta (np. Chrome, Firefox, Safari) i pozwala na interakcję z użytkownikiem, manipulowanie zawartością strony, obsługę zdarzeń, komunikację z serwerem i wiele innych dynamicznych funkcji. JavaScript jest szeroko stosowany zarówno na stronach internetowych, jak i w aplikacjach internetowych, takich jak gry online, narzędzia do analizy danych, społecznościowe platformy, e-commerce i wiele innych. Wiedza z zakresu JavaScriptu jest kluczowa dla wielu programistów, którzy chcą tworzyć interaktywne i zaawansowane aplikacje internetowe.


2. Zmienne

Zmienne w JavaScript są podstawowymi elementami języka, które służą do przechowywania i manipulowania danymi. W JavaScript, deklaracja zmiennych odbywa się za pomocą słowa kluczowego "var", "let" lub "const", a następnie przypisanie wartości do zmiennej odbywa się za pomocą operatora przypisania "=" lub innych operatorów przypisania z dodatkowymi operacjami.


Oto kilka podstawowych informacji na temat zmiennych w JavaScript:


1.Deklaracja zmiennych


  • Za pomocą słowa kluczowego "var": Przed ES6 (ECMAScript 6), "var" było używane do deklaracji zmiennych. Może być zadeklarowane wielokrotnie w tej samej funkcji lub globalnie, a także może zmieniać swoją wartość po zadeklarowaniu.

Przykład:


var x = 5;
var y = "Hello";

  • Za pomocą słowa kluczowego "const": "const" służy do deklarowania zmiennych, których wartość nie może być zmieniona po zadeklarowaniu. "const" ma również zasięg blokowy, podobnie jak "let".

    • Przykład:


    const x = 5;
const y = "Hello";

    2.Przypisanie wartości:


    Przypisanie wartości do zmiennej odbywa się za pomocą operatora przypisania "=".


    Przykład:


    let x = 5;
x = 10;

    3.Korzystanie z zmiennych:


  • Zmienne mogą być używane do przechowywania różnego rodzaju danych, takich jak liczby, łańcuchy znaków, obiekty, tablice i inne.

    • Przykład:


    let x = 5;
let y = "Hello";
let z = [1, 2, 3];

    4.Zasięg zmiennych:


    • "var" ma zasięg funkcyjny, co oznacza, że jest dostępne w obrębie funkcji, w której zostało zadeklarowane, lub globalnie, jeśli jest zadeklarowane poza funkcją.

    • "let" i "const" mają zasięg blokowy, co oznacza, że są dostępne tylko w obrębie bloku, w którym zostały zadeklarowane (np. pętla, instrukcja warunkowa).

    3.Tworzenie stringów:

    Stringi w JavaScript można tworzyć za pomocą pojedynczych lub podwójnych cudzysłowów.


    let str1 = 'Hello';
let str2 = "World";

    Stringi można łączyć (konkatenować) za pomocą operatora "+".


    let fullName = "John" + " " + "Doe";

    Aby uzyskać długość stringa, można użyć właściwości "length".


    let str = "Hello";
console.log(str.length); // wyświetli 5

    Metody do pracy ze stringami:


    • JavaScript posiada wiele wbudowanych metod do pracy ze stringami, takich jak:
    • "toUpperCase()" - zamienia string na wielkie litery
    • "toLowerCase()" - zamienia string na małe litery
    • "charAt(index)" - zwraca znak o określonym indeksie
    • "substring(startIndex, endIndex)" - zwraca fragment stringa o określonym zakresie indeksów
    • "indexOf(substring)" - zwraca indeks pierwszego wystąpienia określonego podstringa
    • "replace(oldValue, newValue)" - zamienia wszystkie wystąpienia określonej wartości na nową wartość
    • "split(separator)" - dzieli stringa na tablicę na podstawie określonego separatora

    let str = "Hello World";
console.log(str.toUpperCase()); // wyświetli "HELLO WORLD"
console.log(str.indexOf("World")); // wyświetli 6
console.log(str.replace("Hello", "Hi")); // wyświetli "Hi World"

    Template strings:


    Od ES6, można używać tzw. "template strings" lub "template literals", które pozwalają na bardziej zwięzłe tworzenie stringów z wyrażeniami JavaScript wewnątrz nich. Template strings są oznaczane znakiem "`" (backtick) i umożliwiają użycie zmiennych i wyrażeń JavaScript wewnątrz ${...}.


    let name = "John";
let age = 30;
console.log(`My name is ${name} and I'm ${age} years old.`); // wyświetli "My name is John and I'm 30 years old."

    4. Pętla for

    Oto pełna składnia pętli "for" w języku JavaScript:

                                    for (inicjalizacja; warunek; krok) {
    // kod do powtórzenia
}

    Gdzie:

    • "inicjalizacja" - jest opcjonalnym krokiem, który wykonuje się tylko raz, przed rozpoczęciem pętli. Jest to miejsce, w którym można zadeklarować i zainicjalizować zmienne, które będą używane w pętli.
    • "warunek" - jest wyrażeniem logicznym, które określa warunek, który musi być spełniony, aby pętla kontynuowała działanie. Jeśli warunek jest prawdziwy, kod wewnątrz pętli będzie się wykonywać, a jeśli jest fałszywy, pętla zostanie zakończona.
    • "krok" - jest opcjonalnym krokiem, który wykonuje się po każdej iteracji pętli, przed sprawdzeniem warunku. Może to być np. zwiększenie lub zmniejszenie zmiennej kontrolującej pętlę.

    Przykład użycia pętli for:

                                    for (let i = 0; i < 5; i++) {
    console.log(i);
}

    W tym przykładzie pętla "for" zostanie wykonana pięć razy, zaczynając od wartości "0" dla zmiennej "i", dopóki "i" jest mniejsze od "5". Wewnątrz pętli wypisuje wartości zmiennej "i" na konsoli, zwiększając ją o jeden po każdej iteracji.

    Pętla "for" jest bardzo przydatnym narzędziem do iteracji przez kolekcje, takie jak tablice, a także do powtarzania określonego bloku kodu określoną liczbę razy. Można ją również zagnieżdżać w innych pętlach lub używać w bardziej zaawansowanych konstrukcjach.
