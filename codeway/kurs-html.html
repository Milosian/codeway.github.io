Kurs HTML - wszystko co musisz wiedzieć, aby zacząć

1. Trochę słów o HTML

HTML, czyli HyperText Markup Language, to język służący do tworzenia struktur i zawartości stron internetowych. HTML definiuje różne elementy, takie jak nagłówki, paragrafy, obrazy i linki, a także umożliwia tworzenie formularzy, tabel i innych elementów interaktywnych na stronie internetowej. Kod HTML składa się z tagów (znaczników), które określają, jak dany fragment tekstu lub element powinien być interpretowany przez przeglądarkę internetową. Przy użyciu HTML można również tworzyć linki, formularze, tabele i wiele innych elementów. Jest to podstawowy język używany przy tworzeniu stron internetowych i jest często łączony z innymi technologiami, takimi jak CSS (arkusze stylów) i JavaScript, aby uzyskać bardziej zaawansowane i interaktywne strony internetowe.


2. Struktura kodu

Poniżej przykładowa struktura kodu:

                                      
<!DOCTYPE HTML>
<html lang="pl">
    <head>
        <meta charset="utf-8"/>
        <meta name="description" content="" />
        <meta name="keywords" content="" />
        <title>Tytuł strony</title>
    </head>
    <body>
        <header>
            <h1>Nagłówek h1</h1>
        </header>
                                
        <nav>Nawigacja</nav>
                                
        <main>
            <section>
                <article>
                    <h2>Artykuł pierwszej w nagłówku h2</h2>
                    <p>To jest treść artykułu.</p>
                </article>
            </section>
        </main>
                                
        <footer>
        Stopka
        </footer>
    </body>
</html>

Wyjaśnienie poszczegółnych znaczników:


  • html - określa, że strona jest dokumentem HTML.
  • head - zawiera informacje o dokumencie, takie jak tytuł strony, meta tagi, styl CSS itp. Informacje te nie są wyświetlane w oknie przeglądarki.
  • meta - definiuje informacje o dokumencie (opis strony, kodowanie znaków, słowa kluczowe itp.). Znacznik ten pomaga wyszukiwarkom internetowym w pozycjonowaniu strony
  • title - określa tytuł strony internetowej, który jest wyświetlany na pasku tytułu przeglądarki.
  • body - określa zawartość strony, która jest wyświetlana w przeglądarce internetowej. Wewnątrz elementu body znajdują się wszystkie elementy strony, takie jak nagłówki, tekst, obrazy, linki itp.
  • header - nagłówek strony, który zazwyczaj zawiera logo, menu nawigacyjne i inne informacje na temat strony.
  • main - definiuje główną treść strony, która jest zazwyczaj wyświetlana w centrum strony.
  • nav - określa menu nawigacyjne na stronie internetowej.
  • section - sekcja strony internetowej, taka jak wpis na blogu, artykuł, galeria zdjęć itp.
  • article - pojedynczy artykuł lub wpis na stronie internetowej.
  • footer - zawiera informacje o autorze, prawach autorskich, linki do stron powiązanych itp.

3. Znaczniki

Znacznik jest to specjalny tekst umieszczony w nawiasach ostrych. Jest częścią składni języka HTML i pozwala sterować wyglądem strony. Dzięki niemu możesz np. ustalić kolor tła, rodzaj formatowania tekstu, wstawić obrazek czy tabelę itd.

Akapit

Akapit zawsze zaczyna się od nowej linii i jest zazwyczaj blokiem tekstu.

                                <p>To jest paragraf</p>
<p>To też jest paragraf</p>

Pogrubienie tekstu

Znacznik pogrubionego tekstu sprawi, że Twój tekst zostanie wyświetlony jako pogrubiony.

                                <b>Pogrubiony tekst</b>
Normalny tekst

Pochylenie tekstu - kursywa

Ten znacznik umożliwi Ci utworzenie tekstu pochylonego

                                <i>Pochylony tekst</i>
Normalny tekst

Podkreślenie tekstu

Znacznik podkreslenia sprawi, że tekst jest podreślony

                                <u>Podkreślony tekst</u>
Normalny tekst

Wstawianie obrazka

By wstawić zdjęcie, należy zastosować znacznik img.

                                <img src="ścieżka dostępu do obrazka" alt="Tu podaj tekst alternatywny">

Odsyłacze

Odsyłacze praktycznie znajdują się na każdej stronie. Może być on zawarty w wyróżnionym hasłowym tekście, bądź w formie obrazka.


Poniższy znacznik odsyła do podstrony: 
                                <a href="ścieżka dostępu do podstrony">opis odsyłacza</a>

Istnieją także odsyłacze do adresu internetowego

                                <a href="adres internetowy">opis odsyłacza</a>

Nagłówki

Znaczniki h1, h2, h3, h4, h5, h6 są stosowane do oznaczenia nagłówka tekstu. Dla przykładu, tytuł artykułu na blogu będzie oznaczony h1 a podtytuł h2. Takie stopniowanie ważności kończy się na znaczniku h6. 
                                <h1>Nagłówek 1. stopnia</h1>
<h2>Nagłówek 2. stopnia</h2>
<h3>Nagłówek 3. stopnia</h3>
<h4>Nagłówek 4. stopnia</h4>
<h5>Nagłówek 5. stopnia</h5>
<h6>Nagłówek 6. stopnia</h6>

Tabele

Tabele są obszarem, w którym można przedstawić jakieś dane.

Przykładowa struktura tabeli:

                                <table>
<tr>
    <td>komórka1</td>	<td>komórka2</td>
    </tr>
<tr>
    <td>komórka3</td>	<td>komórka4</td>
</tr>
</table>

Formularze

Znacznik form jest wykorzystywany do stworzenia formularza. Znacznik ten same w sobie nie definiuje pól formularza takich jak "Imię" czy "Nazwisko", ale jest rodzicem dla tych pól. Najważniejszymi atrybutami znacznika form są action i method.

                                <form action="/send" method="post">
   Imię: <input type="text" name="imie">
   Nazwisko: <input type="text" name="nazwisko">
   <input type="submit" value="Wyślij">
</form>
